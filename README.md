# Moodboard – Portafolio Web Agustín Avendaño  
*(2025)*

## Objetivo del Portafolio

Mi portafolio tiene como propósito presentar mi trabajo de manera profesional y clara, con el objetivo de potenciar mis oportunidades laborales y construir una identidad visual sólida como diseñador. Quiero mostrar una selección de proyectos realizados tanto en la universidad como en encargos personales y colaboraciones externas, organizados bajo un lenguaje visual coherente, moderno y editorial.  
Este portafolio está dirigido a posibles empleadores, agencias, estudios de diseño y profesionales del área que buscan evaluar mis capacidades en diseño gráfico, UX/UI, branding, producto y visualización 3D.

### Lo que me gusta  
- El menú lateral izquierdo minimalista y su despliegue limpio.  
- El panel que aparece al abrir ese menú: tipografía grande, aire y estructura editorial.  
- La barra de progreso vertical en el lado derecho que indica avance o cambio de proyecto.  
- La simpleza del layout y el uso inteligente del espacio negativo.  
- El ritmo entre texto e imagen sin saturación.

### A tener cuidado  
- Evitar copiar su estilo exacto; debo mantener una identidad propia.  
- El minimalismo extremo puede restar fuerza a mis proyectos si no se balancea correctamente.  
- Usar el menú lateral con precaución para que no se convierta en un elemento pesado o repetitivo.

- Las transiciones y animaciones sutiles que acompañan el scroll.

- No saturar el sitio con animaciones excesivas.

## 6. Interfaz editorial y estructura limpia  
**Referentes combinados:** Alex Coven, BREND, Vallon Emma

### Imagen sugerida  
Usar capturas propias una vez que el diseño esté maquetado.

### Lo que me gusta  
- Un enfoque editorial moderno combinado con UI minimalista.  
- El uso de tipografía como protagonista.  
- Un ritmo visual limpio, profesional y versátil.

### A tener cuidado  
- La coherencia entre proyectos muy diferentes (editorial, industrial, UX/UI, 3D).  
- No caer en un diseño demasiado frío o excesivamente artístico.  
- Mantener un color de acento controlado y funcional.

## Notas personales basadas en mi portafolio actual  
Mi portafolio contiene proyectos universitarios, colaboraciones con marcas reales, trabajos de UX/UI, industrial y gráfico. Este nivel de variedad necesita:

- una base visual neutra y editorial  
- una tipografía sans serif con familia amplia  
- una estructura de navegación clara  
- un sistema de portadas consistente  
- animaciones sutiles que entreguen dinamismo sin distraer  
- una identidad visual que unifique todo el contenido

Con este moodboard establezco la dirección visual que guiará el diseño final de mi portafolio web.
